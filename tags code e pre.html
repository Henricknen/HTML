O codigo a seguir representa o corpo basico de um programa em C.

              
#include ‹studio.h›
#include ‹stdlib.h›       
#include ‹locale.h›

int main() {

    setlocale(LC_ALL,"Portuguese");
    printf("Codigo basico em C");

    return 0;
}

Os cabeçalhos ultilizados podem variar de acordo com os recursos que deseja ultilizar no programa, então, stdlib e locale podem ser omitidos como parte do codigo basico, assim como a função setlocale.
